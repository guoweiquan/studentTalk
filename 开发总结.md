## 下一步操作

### 1. 启动数据库
```bash
cd d:\Code_Repository\studentTalk
docker-compose up -d mysql
```

### 2. 启动后端
```bash
cd d:\Code_Repository\studentTalk\backend
npm install
npx prisma generate
npm run dev
```

### 3. 启动前端 (H5 模式测试)
```bash
cd d:\Code_Repository\studentTalk\frontend
npm install
npm run dev:h5
```

### 4. 微信小程序开发
```bash
npm run dev:mp-weixin
# 使用微信开发者工具打开 dist/dev/mp-weixin 目录
```
